# 🎮 FocusRank

## 🚀 Overview

**FocusRank** is a gamified productivity web app for students. Instead of just completing tasks, users **earn XP**, **level up**, and **climb ranks**—turning study sessions into a rewarding loop.

**Core goals**:

* Reduce procrastination with micro-tasks
* Increase consistency with streaks
* Reinforce focus using Pomodoro
* Provide clear progress via stats & visuals

---

## ✨ Key Features

### 📋 Task System

* Create an objective (e.g., *Review Math Chapter 5*)
* Auto-generate actionable steps
* Interactive checklist with persistence (localStorage)
* Real-time progress (e.g., 2/4 completed)

### ✂️ Micro‑Task Mode

* Break down difficult tasks into smaller steps
* One-click “Too hard?” decomposition

### ⚡ XP & Levels

* +10 XP per task
* +25 XP per Pomodoro session
* +50 XP for daily goal completion
* Animated XP bar with progressive leveling curve

### 🏆 Ranks

* Bronze → Silver → Gold → Platinum → Diamond → Master
* Rank evolves based on level

### 🔥 Streak

* Tracks consecutive active days
* +10% XP bonus when streak is active

### 🎯 Daily Goal

* Set targets (tasks or sessions)
* Progress tracking with bonus XP on completion

### ⏱️ Pomodoro Timer

* 25/5 and 50/10 modes
* Start / Pause / Reset
* End-of-session notifications
* Circular progress animation

### 📊 Statistics

* Time studied (today)
* Tasks completed
* Pomodoro sessions
* Daily activity charts (Chart.js)

### 🎯 Focus Mode

* Minimal UI
* Shows only current task + timer

### 🎨 Visual Feedback

* Smooth animations (CSS transitions)
* XP gain effects
* Dynamic colors for states (progress/focus)
